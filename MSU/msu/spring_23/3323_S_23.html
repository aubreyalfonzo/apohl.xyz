CURRENT PROJECT(S)

PROJECT 2 / GRAPHIC MANUAL / MSU-REBRAND

This project will approach two objectives simultaneously:

1. To re-design completely the visual identity system of a selected Mississippi State University Academic Unit (see list).

2. To document this brand via the editorial design of a graphic manual (presentation).

Each student will pick a MSU academic unit from the provided list, and after a period of research and concept development, completely redesign the brand and identity for the selected department as if the current MSU identity system had never existed.
